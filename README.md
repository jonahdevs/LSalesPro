composer install

---

### 3. Set up database
- Set your database values in the .env file
DB_DATABASE=lsalespro
DB_USERNAME=root

### 4. Run Migrations and Seeder
php artisan migrate

### 5. Start the Server
- you can start your server using one of the following

php artisan serve
